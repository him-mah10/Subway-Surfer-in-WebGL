WELCOME TO THE GAME


The objective of the game is to reach the final line by collecting maximum coins and avoiding obstacles.

Controls:

  1. Use a/d keys to change the track
  2. Use space bar to jump
  3. Use g key to toggle between grayscale and colour
Click on the play button to start the game. It will take 5-10 seconds for the game to start
